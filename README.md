# 探测分包撮合平台 - HTML交互原型

## 项目简介

这是一个探测分包撮合平台的HTML交互原型，用于展示小程序的功能和交互流程。原型完全基于HTML + CSS + JavaScript开发，无需后端服务器，可直接在浏览器中打开查看。

**设计特点：**
- 📱 移动端适配（响应式设计）
- 🎨 智能入口系统（记忆用户选择）
- 🔗 页面间可点击跳转
- 💡 功能完整，流程清晰
- 🎯 双端设计（甲方蓝色主题 + 乙方橙色主题）

## 核心功能

### 智能入口系统

- **首次访问**：显示角色选择弹窗（包工头 vs 发包方）
- **记忆功能**：使用localStorage记住用户选择
- **自动跳转**：再次访问时自动进入上次选择的端口
- **角色切换**：在各自首页可随时切换角色

### 甲方端（发包方）- 蓝色主题

**主要功能：**
- 🏠 **首页**：发布项目、包工头库、我的项目
- 📝 **发布项目**：详细的项目发布表单
- 📋 **我的项目**：管理发布的项目和投标
- 👥 **包工头库**：浏览和邀请专业包工头
- 📂 **项目分类**：按类别查看项目类型
- 💬 **消息中心**：项目消息和系统通知
- 👤 **个人中心**：企业资料、资金管理、合同管理
- ⚙️ **设置**：通知设置、账户安全、帮助中心

### 乙方端（包工头）- 橙色主题

**主要功能：**
- 🏠 **首页**：搜索项目、服务分类、热门项目
- 📂 **项目分类**：按类别浏览可投标项目
- 💬 **消息中心**：投标通知、项目消息、私信
- 👤 **个人中心**：个人资料、我的项目、收入统计
- ⚙️ **设置**：通知设置、账户安全、帮助中心

## 快速开始

### 方法一：本地浏览

1. 下载整个项目文件夹
2. 双击打开 `index.html` 文件
3. 首次访问会弹出角色选择弹窗
4. 选择身份后进入对应首页

### 方法二：手机预览

1. 将项目文件夹部署到Web服务器
2. 用手机浏览器访问服务器地址
3. 获得真实的移动端体验

### 方法三：本地HTTP服务器

使用Python快速启动本地服务器：

```bash
# 进入项目目录
cd 探测分包项目

# Python 3
python -m http.server 8000

# 或 Python 2
python -m SimpleHTTPServer 8000
```

然后在浏览器访问：`http://localhost:8000`

## 交互说明

### 智能入口系统

- **首次访问**：显示品牌加载动画 → 弹出角色选择弹窗
- **角色选择**：选择后保存到localStorage并跳转
- **再次访问**：自动跳转到上次选择的端口
- **角色切换**：在首页右上角可随时切换

### 页面跳转

- **底部导航栏**：切换主要功能模块
- **角色切换按钮**：在头部模块内，可切换用户端
- **功能按钮**：跳转到对应功能页面
- **返回按钮**：返回上一页

### 响应式设计

- **移动端优先**：专为手机屏幕设计
- **自适应布局**：在不同屏幕尺寸下正常显示
- **触摸友好**：按钮大小适合手指点击

## 技术实现

### 前端技术

- **HTML5**：语义化标签，结构清晰
- **CSS3**：Flexbox布局，渐变背景，过渡动画
- **JavaScript**：localStorage记忆功能，页面跳转逻辑

### 核心功能实现

- **智能入口**：localStorage + 条件跳转
- **角色切换**：动态保存用户选择
- **响应式布局**：CSS媒体查询 + Flexbox
- **交互反馈**：CSS过渡动画 + 悬停效果

## 部署说明

### 静态部署

项目为纯静态HTML页面，可部署到任何Web服务器：

- **GitHub Pages**：直接上传到GitHub仓库
- **Vercel**：拖拽文件夹即可部署
- **Netlify**：连接Git仓库自动部署
- **腾讯云COS**：上传静态文件到对象存储
- **阿里云OSS**：静态网站托管

### 服务器部署

1. 将项目文件上传到服务器
2. 配置Web服务器（Nginx/Apache）
3. 确保index.html为默认首页
4. 访问域名即可使用

## 注意事项

⚠️ **这是一个前端原型，不包含后端功能：**
- 所有数据都是静态示例
- 表单提交不会真正保存
- 页面刷新后部分状态会重置
- localStorage数据会持久保存

💡 **最佳使用方式：**
- 使用Chrome浏览器打开（F12开启移动端模式）
- 或在实际手机浏览器中查看
- 按照智能入口流程体验完整功能
