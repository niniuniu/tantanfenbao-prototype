# 探测分包撮合平台 - 微信小程序原型设计方案

## 设计风格定位
- **色彩基调**：以深蓝色（#1E3A8A）为主色，象征专业与信任；辅以橙色（#F59E0B）作为强调色
- **风格关键词**：专业、稳重、高效、可信赖
- **字体**：使用微信默认字体，标题加粗，正文常规
- **图标风格**：线性图标为主，简洁明了

## 五、交互流程说明

### 5.1 甲方发布需求流程
```
首页 → 点击"发布需求" → 填写项目信息 → 设置资质要求 
→ 设置投标规则 → 上传附件 → 提交发布 → 等待投标 
→ 查看投标列表 → 在线开标 → 选择中标方 → 签署合同 
→ 查看进度 → 阶段验收 → 付款 → 最终验收 → 评价
```

### 5.2 乙方投标接单流程
```
找活首页 → 查看项目详情 → 判断是否符合要求 → 点击投标 
→ 填写报价 → 上传资质证明 → 支付保证金 → 提交投标 
→ 等待开标 → 中标通知 → 签署合同 → 开始施工 
→ 上报进度 → 申请验收 → 提交成果 → 申请结算 
→ 收到款项 → 评价甲方
```

### 5.3 平台资金托管流程
```
甲方充值 → 发布需求（冻结部分资金） → 乙方中标 
→ 甲方首付款（平台托管） → 阶段验收通过 
→ 平台自动划款给乙方（扣除平台佣金） → 最终验收 
→ 尾款支付 → 平台分佣 → 完成交易
```

## 六、设计规范

### 6.1 颜色规范
- **主色**：#1E3A8A（深蓝色）- 用于标题、按钮、重点信息
- **辅助色**：#F59E0B（橙色）- 用于强调、提示、激活状态
- **成功色**：#10B981（绿色）- 用于成功状态、完成标识
- **警告色**：#F59E0B（橙色）- 用于警告提示
- **错误色**：#EF4444（红色）- 用于错误、危险操作
- **文本色**：
  - 主文本：#1F2937
  - 次要文本：#6B7280
  - 禁用文本：#D1D5DB
- **背景色**：
  - 页面背景：#F9FAFB
  - 卡片背景：#FFFFFF
  - 分割线：#E5E7EB

### 6.2 字体规范
- **大标题**：18px / Bold / #1F2937
- **标题**：16px / Bold / #1F2937
- **正文**：14px / Regular / #1F2937
- **辅助文字**：12px / Regular / #6B7280
- **按钮文字**：16px / Medium / #FFFFFF

### 6.3 间距规范
- **页面边距**：16px
- **卡片间距**：12px
- **元素间距**：8px
- **小间距**：4px

### 6.4 圆角规范
- **按钮圆角**：8px
- **卡片圆角**：12px
- **输入框圆角**：6px
- **图片圆角**：4px

### 6.5 按钮规范
- **主要按钮**：高度48px，背景#1E3A8A，文字白色，圆角8px
- **次要按钮**：高度48px，背景白色，边框1px #1E3A8A，文字#1E3A8A
- **警告按钮**：高度48px，背景#EF4444，文字白色
- **文字按钮**：无背景，文字#1E3A8A

### 6.6 状态标识
- **进行中**：🟢 绿色圆点 + 文字
- **待处理**：🔵 蓝色圆点 + 文字
- **已完成**：✅ 或 ⚪ + 灰色文字
- **已取消**：❌ + 灰色文字

## 十、注意事项

1. **合规性**：
   - 建筑资质审查需符合国家规定
   - 资金托管需有相关金融牌照或合作持牌机构
   - 电子合同需符合《电子签名法》

   - 用户隐私保护
   - 资金交易安全
   - 数据传输加密

3. **用户体验**：
   - 操作流程简洁明了
   - 关键信息突出显示
   - 及时的状态反馈

4. **性能优化**：
   - 图片压缩与懒加载
   - 列表分页加载
   - 缓存策略
